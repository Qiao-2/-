# 会员专属个人站

这是一个适合 Vercel 部署的轻量版登录/注册/管理员审核模板。

## 功能
- 用户注册
- 管理员审核通过/拒绝
- 审核通过后才能登录
- 登录后进入内部页面
- 管理员后台审核列表

## 本地运行
```bash
npm install
npm start
```

然后打开 `http://127.0.0.1:3000`。

## Vercel 部署
1. 把整个文件夹推到 Git 仓库
2. 在 Vercel 导入这个仓库
3. 保留默认框架识别即可
4. 部署完成后直接访问生成的项目地址

## 默认管理员
- 用户名：`admin`
- 密码：`123456`
